# User Personas

## Persona: FFXI Multiboxer

- **Name**: The Multiboxer
- **Description**: An FFXI player who runs multiple game clients simultaneously to fill party or alliance roles with their own characters.
- **Technical Level**: Comfortable running command-line tools, editing config files, managing batch scripts. Not necessarily a developer.
- **Goals**:
  - Launch multiple FFXI accounts quickly with minimal manual interaction
  - Avoid repetitive password entry and POL navigation
  - Switch between different team compositions easily
  - Keep credentials secure on their local machine
- **Frustrations**:
  - POL's 4-account limit per login_w.bin forces manual file swapping
  - Existing batch scripts require hardcoded paths and are fragile
  - Forgetting which accounts are in which backup file
  - Having to manually time launches and file swaps
- **Environment**: Windows PC, typically with Windower installed, running 2-18 game clients. Admin access available.
